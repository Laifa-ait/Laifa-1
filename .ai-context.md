# Contexte IA — OLMART

## Regles de Code
- TypeScript strict, pas de `any`
- React 19 + hooks (pas de classes)
- Tailwind CSS v4 (pas de tailwind.config.js, utiliser @theme)
- Framer Motion pour les animations
- Lucide React pour les icones
- i18next pour les traductions (fr, ar, en)

## Architecture
- Components: src/components/ (atomiques, réutilisables)
- Pages: src/pages/Public/, src/pages/Seller/, src/pages/Admin/
- Hooks: src/hooks/ (logique métier réutilisable)
- Utils: src/utils/ (fonctions pures)

## Ne JAMAIS faire
- Importer `firebase` dans un composant (utiliser les hooks/services)
- Utiliser `window.alert` ou `window.confirm`
- Laisser des `console.log` en production
- Utiliser des magic numbers (toujours des constantes)
